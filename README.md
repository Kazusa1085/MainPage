# Kazusa1085 Main Page

**感谢[@moewah](https://github.com/moewah)大佬的[MoeHome](https://github.com/moewah/MoeHome)项目**

该项目是MoeHome项目的Astro重构，用于学习Astro。

![License](https://img.shields.io/badge/License-MIT-green)

## 快速开始

### 安装

```bash
git clone https://github.com/Kazusa1085/MainPage.git
cd MainPage
npm install  # 安装依赖
```

### 开发

```bash
npm run build
# 本地预览
npm run preview
# 访问 http://localhost:43210
```
